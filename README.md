# Vectroid

## How to Play

Survive as long as possible by defeating enemies and collecting XP orbs.

## Controls

### Keyboard
- **Shift** - Dash (speed boost with cooldown)
- **Escape** - Pause/Unpause
- **R** - Restart game

### Gamepad
- **Left Stick** - Move and face direction
- **Right Stick** - Look direction
- **Right Trigger** - Dash
- **Start Button** - Pause/Unpause/Restart
- **D-Pad Up/Down** - Navigate upgrade menu
- **A / X Button** - Select upgrade

## Running the Game

Simply open `index.html` in a modern web browser.

## Browser Compatibility

Works best in modern browsers that support:
- HTML5 Canvas
- ES6 JavaScript
- Web Audio API (for sound effects)
- Gamepad API (for controller support)
